# Phone Contact UI (JavaFX)

A simple desktop-based Phone Contact Manager built using **JavaFX**.  
This application allows users to add, update, delete, and manage contacts with name, phone number, and email.

---

## 📌 Features

- Add new contacts (Name, Phone, Email)
- Update selected contact
- Delete selected contact
- Clear input fields
- Display contacts in a TableView
- Click a row to auto-fill fields for editing

---

## 🛠️ Technologies Used

- Java
- JavaFX
- ObservableList (for dynamic table data)
- TableView (for UI display)
---

## 🚀 How to Run

### 1. Requirements
- JDK 8 or higher (JavaFX included OR configured separately)
- IDE (NetBeans / IntelliJ / Eclipse) or terminal setup with JavaFX

### 2. Steps
#### Using IDE:
1. Open the project in your IDE
2. Make sure JavaFX libraries are added (if required)
